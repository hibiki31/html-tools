# 海洋マッピング座標計算ツール

## 概要

基準点（海抜0m）を原点とし、距離・深度・方角から海中または海上のマッピング地点の x, y, z 座標を計算するツールです。

```
x : 東西方向（+東 / -西）
y : 南北方向（+北 / -南）
z : 深度方向（-深度）
```

## 操作方法

1. ポイント名称・アイコンを設定する
2. 距離・深度・方角・方角の意味・距離の種類を入力する
3. 「計算する」ボタンを押すか、入力値が変わると自動的に計算される
4. 「履歴と地図へ追加」でマップに確定登録する
5. 「CSV保存」で履歴をファイルに書き出す

### マップ操作

| 操作 | 内容 |
|------|------|
| マウスホイール | ズームイン / アウト（カーソル位置中心） |
| ドラッグ | マップをパン（移動） |
| タッチ1本指 | パン |
| タッチ2本指（ピンチ） | ズームイン / アウト |
| 「地図を再描画」ボタン | ズーム・パンをリセットして全体表示 |

## 実装済み機能

| 機能 | 説明 |
|------|------|
| 座標計算 | 距離・深度・方角からx, y, zを算出 |
| 斜距離 / 水平距離切替 | 入力距離の種類に応じて水平距離を計算 |
| 方角変換 | 「マッピング→基準」「基準→マッピング」の2モード |
| コンパス表示 | 方角をSVGコンパスで視覚化 |
| 計算式表示 | 計算ステップを逐次表示 |
| ポイント名称指定 | 任意のポイント名称を設定 |
| ポイントアイコン | 24種類の海洋ゲーム向け絵文字から選択 |
| 自動カラー割り当て | ポイントごとに10色パレットから自動割り当て |
| マップ表示 | SVGによる2D俯瞰マップ |
| マップ パン / ズーム | ホイール・ドラッグ・タッチ操作 |
| 動的グリッド | スケールに応じてラベル間隔を自動調整（重なり防止） |
| プレビュー | 未確定ポイントを点線でリアルタイムプレビュー |
| 計算履歴 | 登録ポイントをテーブルで一覧表示 |
| 個別削除 / 全削除 | 履歴からポイントを削除 |
| 結果コピー | 計算結果をクリップボードへコピー（成功/失敗をトースト通知） |
| CSV保存 | UTF-8 BOM付きCSVでエクスポート |
| CSV読み込み | 不正行をスキップし件数を通知（バリデーション付き） |

## グリッド仕様

| 種類 | 基本間隔 | 自動拡大条件 |
|------|----------|-------------|
| 細線（目盛り） | 10m | ピクセル幅 < 3px で2倍 |
| 太線（格子） | 100m | ラベル間隔の半分に固定 |
| 数字ラベル | 200m | ピクセル幅 < 40px で2倍 |

ズームインすれば細かいグリッドが表示され、ズームアウト・遠距離でもラベルが重ならない。

## 技術的注意事項

### 地球曲率について

本ツールは **数百m〜数km程度のローカル座標計算** を想定しています。  
広域測量・高精度海洋調査では以下の考慮が別途必要です：

- 地球曲率
- 測地系（WGS84 等）
- UTM投影座標
- 潮位補正
- 磁北・真北差
- センサー誤差

### 斜距離と水平距離

```
斜距離（slant）：  水平距離 = √(distance² - depth²)  ※ distance ≥ depth が必須
水平距離（horizontal）：水平距離 = distance
```

### 方角変換

```
マッピング地点 → 基準点：bearing_from_base = (input + 180) mod 360
基準点 → マッピング地点：bearing_from_base = input
```
