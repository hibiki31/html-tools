# 海洋マッピング座標計算ツール

## 概要

基準点（海抜0m）を原点とし、距離・深度・方角から海中または海上のマッピング地点の x, y, z 座標を計算するツールです。  
単一HTMLファイルで動作し、外部ライブラリは一切使用しません。

```
x : 東西方向（+東 / -西）
y : 南北方向（+北 / -南）
z : 深度方向（-深度）
```

## 使い方

1. `main.html` をブラウザで開く（ダブルクリック、または `start main.html`）
2. ポイント名称・アイコンを設定する
3. 距離・深度・方角・方角の意味・距離の種類を入力する
4. 「計算する」ボタンを押すか、入力値が変わると自動的に計算される
5. 「履歴と地図へ追加」でマップに確定登録する
6. 「CSV保存」で履歴をファイルに書き出す

### マップ操作

| 操作 | 内容 |
|------|------|
| マウスホイール | ズームイン / アウト（カーソル位置中心） |
| ドラッグ | マップをパン（移動） |
| タッチ1本指 | パン |
| タッチ2本指（ピンチ） | ズームイン / アウト |
| 「地図を再描画」ボタン | ズーム・パンをリセットして全体表示 |

## グリッド仕様

| 種類 | 基本間隔 | 自動拡大条件 |
|------|----------|-------------|
| 細線（目盛り） | 10m | ピクセル幅 < 3px で2倍 |
| 太線（格子） | 100m | ラベル間隔の半分に固定 |
| 数字ラベル | 200m | ピクセル幅 < 40px で2倍 |

ズームインすれば細かいグリッドが表示され、ズームアウト・遠距離でもラベルが重ならない。

## 開発方針

### アーキテクチャ

- **単一HTMLファイル** で完結させる（HTML + CSS + JavaScript）
- 外部CDN・ライブラリは使用しない（オフライン動作・セキュリティのため）
- 地図描画には **SVG** を使用する（ベクタ形式で高精細・拡張しやすい）
- データはブラウザのJavaScript変数に保持（LocalStorage等への永続化は将来拡張）

### コーディング規則

- JavaScript はモダンES2020+を使用（`const`, `let`, アロー関数, テンプレートリテラル等）
- `document.getElementById` は `const $ = (id) => document.getElementById(id)` でラップ
- HTML出力には必ず `escapeHtml()` を使用してXSS対策を行う
- CSV入出力は `csvEscape()` / `parseCsv()` で完結させ、ライブラリに依存しない
- `formatNumber()` を通じて日本語ロケールで数値を表示する

### SVGマップの実装方針

- SVG内部座標は固定（900×560）
- **viewBox** 属性を操作してパン・ズームを実現する（`<g transform>` は使わない）
- `renderMap()` はSVG全体を `innerHTML` で再生成する方式（差分更新は行わない）
- `applyMapViewBox()` を `renderMap()` の末尾で呼び、ズーム状態を保持する

### 命名規則

| 対象 | 規則 |
|------|------|
| 変数・関数 | camelCase |
| HTML id | camelCase |
| CSSクラス | kebab-case |
| CSV列名 | snake_case |

### バグ修正時の注意点

- JavaScript文字列リテラル（`"..."` / `'...'`）内に **実際の改行文字を含めてはならない**  
  → 改行は必ず `\n`、`\r` のエスケープシーケンスで記述すること
- 正規表現リテラル内の改行も同様（`/[",\n\r]/` と書く）
- SVG `innerHTML` に挿入する文字列は必ず `escapeHtml()` を通すこと

## 技術的注意事項

### 地球曲率について

本ツールは **数百m〜数km程度のローカル座標計算** を想定しています。  
広域測量・高精度海洋調査では以下の考慮が別途必要です：

- 地球曲率
- 測地系（WGS84 等）
- UTM投影座標
- 潮位補正
- 磁北・真北差
- センサー誤差

### 斜距離と水平距離

```
斜距離（slant）：  水平距離 = √(distance² - depth²)  ※ distance ≥ depth が必須
水平距離（horizontal）：水平距離 = distance
```

### 方角変換

```
マッピング地点 → 基準点：bearing_from_base = (input + 180) mod 360
基準点 → マッピング地点：bearing_from_base = input
```
